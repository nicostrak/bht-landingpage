¿Cómo funciona?

Integra nuestra API en solo 3 pasos

1

Regístrate y obtén tus credenciales

Crea tu cuenta y accede a tus claves API desde el dashboard.

2

Integra con tu sistema

Usa nuestra documentación para conectar tu software con nuestra API.

3

Envía tus boletas

Emite boletas individuales o en lote directamente al SII.


// Ejemplo de emisión de boleta
const response = await fetch('https://api.boletaapi.com/v1/boletas', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        emisor_rut: '76.543.210-8',
        receptor_rut: '12.345.678-9',
        monto: 250000,
        servicio: 'Asesoría contable',
        fecha: '2023-12-15'
    })
});

const data = await response.json();
console.log(data);
